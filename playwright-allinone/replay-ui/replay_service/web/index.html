        2중 확인을 통과하면 그 결과를 저장해서 이후 시나리오 실행에 자동으로 재사용합니다.
      


      

      
      

        ⚠ 테스트 대상 서비스 진입 페이지를 입력하세요 (네이버 로그인 페이지가 아님).
      


      
      

        시드 후 자동으로 이 URL 에 접근해 로그인 상태를 확인합니다.
      


      
      

        비우면 검증 URL 접근 성공만 확인. 채우면 해당 텍스트까지 보여야 통과 (강한 검증).
      


      

      
      

        시드 후 storage 에 이 도메인의 쿠키가 있어야 합니다. ID/PW 만 쓰는 사이트면 비워 주세요.

    2. 
      
  2. 로그인 화면에서 ID / 비밀번호 입력
    3. 
      
  3. 2중 확인 (SMS / OTP 등) 직접 통과
    4. 
      
  4. 서비스로 자동 복귀 → 검증 대상 페이지 확인
    5. 
      
  5. 로그인 완료 화면 확인 후 열린 브라우저 창을 닫으세요
    6.
