Playwright.dev Zero-Touch QA test scenario

This document is a human-readable companion to the Playwright script used as convert input for zero_touch_qa, capturing the same flow.

Target site
https://playwright.dev
Test purpose
Walk through the official docs site, API reference, and community page in 20 steps.
Run mechanics
Chrome-channel Playwright script → Zero-Touch QA convert → DSL execution.
Preconditions
Internet connectivity; the browser can reach playwright.dev.
Done when
Each step lands on its target page and the specified heading or link is shown.
Step numbers in this document map 1:1 to the DSL steps generated when the Playwright input script is converted by converter.py.

Detailed steps

Step Action Target Description / expected result
