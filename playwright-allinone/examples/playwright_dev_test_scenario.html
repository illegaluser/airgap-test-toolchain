Playwright.dev Zero-Touch QA 테스트 시나리오

본 문서는 zero_touch_qa의 convert 입력용 Playwright 스크립트와 동일한 흐름을 사람이 검토할 수 있도록 정리한 테스트 시나리오 문서다.

대상 사이트
https://playwright.dev
테스트 목적
공식 문서 사이트의 주요 섹션과 API, 커뮤니티 페이지를 20 step으로 순회 검증
실행 방식
Chrome 채널 기반 Playwright 스크립트 → Zero-Touch QA convert → DSL 실행
사전 조건
외부 인터넷 연결 가능, 브라우저에서 playwright.dev 접근 가능
완료 기준
각 단계의 목표 페이지로 이동하고 지정 heading 또는 링크가 화면에 표시됨
문서의 step 번호는 Playwright 입력 스크립트가 converter.py에서 변환될 때 생성되는 DSL step과 1:1로 맞춰져 있다.
