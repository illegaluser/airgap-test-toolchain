Selenium Web Form Zero-Touch QA 테스트 시나리오

본 문서는 외부 사이트 기반으로 9대 DSL 액션을 모두 포함하는 회귀 테스트 예제다. 대응 Playwright 입력 스크립트는 selenium_webform_convert_input.py 이다.

대상 사이트
https://www.selenium.dev/selenium/web/web-form.html
테스트 목적
외부 사이트에서 9대 DSL 액션 전체를 회귀 검증
실행 방식
Playwright convert 입력 또는 PDF doc 입력
완료 기준
폼 입력 후 제출 결과 메시지 Received!가 표시됨

세부 절차

Step 액션 대상 설명 / 기대 결과
1navigatehttps://www.selenium.dev/selenium/web/web-form.html폼 테스트 페이지로 이동한다.
2verifyh1페이지 제목이 Web form 이어야 한다.
3fill#my-text-id텍스트 입력란에 Zero Touch QA를 입력한다.
4press#my-text-idTab 키를 눌러 다음 필드로 포커스를 이동한다.
5filltextarea[name='my-textarea']설명 텍스트를 입력한다.
6selectselect[name='my-select']드롭다운에서 Two를 선택한다.
7check#my-check-2기본 체크박스를 활성화한다.
8hoverbutton제출 버튼 위에 마우스를 올린다.
9wait-400ms 대기한다.
10clickbutton제출 버튼을 클릭한다.
11verify#message결과 메시지가 Received! 여야 한다.
